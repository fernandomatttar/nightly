Security & privacy

Nightly is independent software built to do one small thing well — and to respect your machine and your privacy while doing it.

No data collection

The Nightly app collects no personal data of any kind. There are no accounts and nothing to sign in to.

No internet access

Nightly never connects to the internet. It simply switches a setting that already lives in Windows.

No hidden background process

Nightly Free runs only when you click it, then closes — nothing stays open. Nightly Pro offers an optional tray icon for one-click access; if you enable it, it's the only thing that stays open, it does nothing until clicked, and it never connects to the internet.

No admin rights

Nightly installs per-user, so there's no elevation prompt and no changes to system-wide settings.

The "Windows protected your PC" message

If you see a blue SmartScreen dialog when installing, here's exactly what it means.

Windows SmartScreen shows this screen for newly released independent software that hasn't yet built up a download reputation with Microsoft. It is not a virus alert. Every small, unsigned app starts out this way until enough people have downloaded it — or until the developer buys a code-signing certificate.

To continue safely:

  1. Click More info.
  2. Click Run anyway.

A code-signing certificate for Nightly is planned as the product grows, which will remove this prompt.
